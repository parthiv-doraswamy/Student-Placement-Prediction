# Student Placement Prediction using Machine Learning

## Overview

This project predicts whether a student is likely to be placed based on academic performance, internships, projects, certifications, aptitude scores, and training records.

## Dataset

Public Placement Prediction Dataset

Features:

- CGPA
- Internships
- Projects
- Workshops/Certifications
- Aptitude Test Score
- Soft Skills Rating
- Extracurricular Activities
- Placement Training
- SSC Marks
- HSC Marks

Target:

- Placement Status

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm

- Logistic Regression

## Workflow

1. Data Loading
2. Data Cleaning
3. Label Encoding
4. Train-Test Split
5. Model Training
6. Evaluation
7. Prediction
